Class Watir::TableBodies
In: watir.rb
Parent: ObjectActions

this class is a collection of the table body objects that exist in the table it wouldnt normally be created by a user, but gets returned by the bodies method of the Table object many of the methods available to this object are inherited from the ObjectActions class
