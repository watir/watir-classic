Class Watir::SelectBox
In: watir.rb
Parent: ObjectActions

This class is the way in which select boxes are manipulated. it would not normally be created by a user, as it is returned by the selectBox method of IEController

Methods

clearSelection   getAllContents   getSelectedItems   new   select   select_value  

Public Class methods

new( ieController, how , what )

returns an initialized instance of a SelectBox object 

  * ieController  - an instance of an IEController
  * how         - symbol - how we access the select box
  * what         - what we use to access the select box, name, id etc

Public Instance methods

clearSelection()

This method clears the selected items in the select box

getAllContents()

This method returns all the items in the select list as an array. An empty array is returned if the select box has no contents. Raises UnknownObjectException if the select box is not found

getSelectedItems()

This method returns all the selected items from the select box as an array. Raises UnknownObjectException if the select box is not found.

select( item )

This method selects an item, or items in a select box. Raises NoValueFoundException if the specified value is not found. 

 * item   - the thing to select, string, reg exp or an array of string and reg exps
select_value( item )

This method selects an item, or items in a select box. Raises NoValueFoundException if the specified value is not found. 

 * item   - the value of the thing to select, string, reg exp or an array of string and reg exps
