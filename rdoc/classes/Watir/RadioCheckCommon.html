Class Watir::RadioCheckCommon
In: watir.rb
Parent: ObjectActions

This class is the class for radio buttons and check boxes. It contains methods common to both. It should not be created by users.

isSet?()

This method determines if a radio button or check box is set. Returns true is set/checked or false if not set/checked. Raises UnknownObjectException if its unable to locate an object.
