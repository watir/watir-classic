Class Watir::Link
In: watir.rb
Parent: ObjectActions

This class is the means of accessing a link on a page It would not normally be used bt users, as the link method of IEController would returned an initialised instance of a link. many of the methods available to this object are inherited from the ObjectActions class

Methods

href   innerText   link_has_image   link_string_creator   new   src   text   to_s   type   value  

Public Class methods

new( ieController, how , what )

Returns an initialized instance of a link object 

  * ieController  - an instance of an IEController
  * how         - symbol - how we access the link
  * what         - what we use to access the link, text, url, index etc

Public Instance methods

href()

returns the url the link points to

innerText()

returns the text displayed by the link

link_has_image()

if an image is used as part of the link, this will return true

link_string_creator()
src()

this method returns the src of an image, if an image is used as part of the link

text()

Alias for innerText

to_s()

returns a textual description of the link

type()

returns ‘link’ as the object type

value()

links dont support value, so returns an empty string
