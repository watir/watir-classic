Class Watir::ObjectActions
In: watir.rb
Parent: Object

This class is the base class for most actions ( such as "click ", etc. ) that occur on an object. This is not a class that users would normally access.

Constants

Public Class methods

new( o )

Creates an instance of this class. 

  o  - the object that watir is using

Public Instance methods

click()

This method clicks the active element. raises: UnknownObjectException if the object is not found ObjectDisabledException if the object is currently disabled

disabled()

returns true if the object is disabled

enabled?()

This method returns true if the current element is enable, false if it isn’t. 

  raises: UnknownObjectException  if the object is not found
exists?()

This methods checks to see if the current element actually exists.

fireEvent(event)

This method executes a user defined "fireEvent" for objects with JavaScript events tied to them such as DHTML menus. 

  usage: allows a generic way to fire javascript events on page objects such as "onMouseOver", "onClick", etc.
  raises: UnknownObjectException  if the object is not found
          ObjectDisabledException if the object is currently disabled
fire_event(event)

Alias for fireEvent

flash()

causes the object to flash. Normally used in IRB when creating scripts

focus()

This method sets focus on the active element. 

  raises: UnknownObjectException  if the object is not found
          ObjectDisabledException if the object is currently disabled
getOLEObject()

returns the Object in its OLE form, allowing any methods of the DOM that Watir doesnt support to be used

highLight( setOrClear )

This method is responsible for setting and clearing the colored highlighting on the currently active element. use :set to set the highlight 

  :clear  to clear the highlight
id()

returns the id of the object

innerHTML()

returns the innerHTML of the object as a string - see msdn.microsoft.com/workshop/author/dhtml/reference/properties/innerhtml.asp?frame=true

name()

returns the name of the object as a string

outerHTML()

returns the outer html of the object - see msdn.microsoft.com/workshop/author/dhtml/reference/properties/outerhtml.asp?frame=true

string_creator()

Returns an array with many of the properties, in a format to be used by the to_s method

title()

returns the value of the title attribute

to_s()

This method displays basic details about the object. Sample output for a button is shown. Raises UnknownObjectException if the object is not found. 

     name      b4
     type      button
     id         b5
     value      Disabled Button
     disabled   true
type()

returns a string with the type of the object, or an empty string if it isnt supported. Many objects override this method anyway.

value()

returns the value of the object
