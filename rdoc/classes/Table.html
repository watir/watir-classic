Table (Class)

In: watir.rb
Parent: ObjectActions

This class is used for dealing with tables. This will not be normally used by users, as the table method of IEController would return an initialised instance of a table.

Methods

columns   new   rows   to_a  

Public Class methods

new( ieController, how , what )

Returns an initialized instance of a table object 

  * ieController  - an instance of an IEController
  * how         - symbol - how we access the table
  * what         - what we use to access the table - id, name index etc

Public Instance methods

columns( rowToUse = 1)

This method returns the number of columns in the table. Raises an UnknownTableException if the table doesn’t exist.

rows()

This method returns the number of rows in the table. Raises an UnknownTableException if the table doesnt exist.

to_a()

This method returns the table as a 2 dimensional array. Dont expect too much if there are nested tables, colspan etc. Raises an UnknownTableException if the table doesn’t exist.
