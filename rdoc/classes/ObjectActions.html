ObjectActions (Class)

In: watir.rb
Parent: Object

This class is the base class for most actions ( click etc ) that happen on an object this is not a class that uses would normally access.

Methods

click   enabled?   exists?   focus   getProperty   highLight   new   setProperties   setProperty   to_s  

Public Class methods

new( o )

creates an instance of this class. The initialize method creates several default properties for the object. these properties are accessed using the setProperty and getProperty methods 

  o  - the object - normally found using

Public Instance methods

click()

this method clicks the active element 

  raises: UnknownObjectException  if the object is not found
          ObjectDisabledException if the object is currently disabled
enabled?()

this method returns true if the current element is enable, false if it isnt 

  raises: UnknownObjectException  if the object is not found
exists?()

this methods checks to see if the current element actually exists

focus()

this methods gives focus to the active element 

  raises: UnknownObjectException  if the object is not found
          ObjectDisabledException if the object is currently disabled
getProperty(name)

this method retreives the value of the specified property 

  * name  - string - the name of the property to get
highLight( setOrClear )

this method is responsible for setting and clearing the highlight on the currently active element use :set to set the highlight 

    :clear  to clear the highlight
setProperties(propertyHash )

this method sets the properties for the object

setProperty(name , val)

this method is used to set a property 

  * name  - string - the name of the property to set
  * val   - string - the value to set
to_s()

this method displays basic details about the object. Sample output for a button is shown raises UnknownObjectException if the object is not found 

     name        b4
     type        button
     id          b5
     value       Disabled Button
     disabled    true
