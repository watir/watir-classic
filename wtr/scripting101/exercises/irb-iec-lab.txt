h2=. Lab 2: Driving Web Applications using IRB

Your instructor has demonstrated how drive the web application using
Ruby and IRB. Now it is your turn. Enter ruby commands into IRB to
drive the Timeclock application. 

Use the "Web Testing with Ruby Cheat Sheet":../../doc/Internet%20Explorer%20Cheat%20Sheet.doc and the WATIR User Guide as references.

We suggest that you execute the following steps:

#   Log into the timeclock server using an existing account.
#   Create a new job.
#   Start the new job.
#   Stop the day.

You will need to have Ruby and the Web Testing with Ruby toolkit
installed on your machine before starting.

h3. Completion

When you have completed this lab, you should be able to:

*         Start IE from IRB
*         View the forms on a page
*         Select a form by name or action
*         View the elements, or controls, in a form
*         Select an element by name or value
*         Enter text in a text field
*         Click a button
