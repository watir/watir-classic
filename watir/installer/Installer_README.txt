NSIS Installer for Watir version 1.4.1
Kingsley Hendrickse @ thoughtworks.com

Double click the watir_installer.exe and follow the wizard.

The installer installs the watir libraries and the autoit dll which are mandatory, but then you can choose
whether to install the documentation, examples and unit tests.

The installer also make optional desktop and start menu links to documentation and an uninstaller.

To uninstall click on the uninstall link from the start menu or from the uninstaller created in the Watir directory.
Everything but the ruby libraries is uninstalled.

----

For later releases you will need to install nullsoft to create new watir installers.
